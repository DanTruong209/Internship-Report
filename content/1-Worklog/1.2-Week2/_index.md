## Week 2 Achievements:

This week focused on expanding AWS knowledge from basic cloud deployment into compute architecture and networking concepts.

I completed Module 3, Module 4, and Module 5 of AWS Cloud Practitioner Essentials. Through these modules, I gained a deeper understanding of different compute approaches, including traditional virtual machines, containers, and serverless computing with AWS Lambda.

The networking module helped me understand how AWS resources are organized inside the Cloud environment. Concepts such as Region, Availability Zone, VPC, Subnet, Security Group, and Network ACL became clearer when I explored them directly on AWS Console.

For hands-on practice, I recreated an EC2 Ubuntu environment, reviewed the Node.js deployment process using PM2, and created a simple Lambda function to compare the differences between continuously running servers and serverless execution.
