## Week 4 Achievements:

This week focused on two important areas for future cloud application development: Artificial Intelligence and AWS Security.

I completed Module 8 and Module 9 of AWS Cloud Practitioner Essentials. Through Module 8, I gained an overview of AI/ML capabilities on AWS, Generative AI concepts, and how data analytics pipelines support intelligent applications.

Module 9 provided deeper knowledge about AWS security principles, including IAM, access control, network protection, data protection, threat detection, and incident response. These concepts were especially important because security is a critical requirement when designing cloud systems.

For hands-on practice, I explored IAM on AWS Console by reviewing Users, Groups, Roles, and Policies. I also checked EC2 Security Groups, analyzed inbound/outbound rules, and adjusted basic SSH and HTTP permissions.
