# Serverless Incident Response Management System (IRMS)

### Serverless Cybersecurity Incident Response Management System

This workshop walks through an end-to-end IRMS implementation on AWS. The content is grouped into seven sections so you can move from setup to deployment and cleanup in a logical order.

#### Workshop outline

1. [Introduction](5.1-Introduction/)
2. [Environment Setup](5.2-Environment-Setup/)
3. [Infrastructure Configuration](5.3-Infrastructure-Configuration/)
4. [Application Development](5.4-Application-Development/)
5. [Deployment and Testing](5.5-Deployment-and-Testing/)
6. [Frontend Integration and Web Deployment](5.6-Frontend-Integration-Web-Deployment/)
7. [Results, Cost Analysis, and Cleanup](5.7-Results-Cost-Analysis-and-Cleanup/)

#### Expected outcome

By the end of the workshop, you should have a working IRMS-style demonstration deployed with serverless services, authentication, storage, automation, and monitoring on AWS.
