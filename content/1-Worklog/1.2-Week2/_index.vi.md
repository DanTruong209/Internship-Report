## Kết quả đạt được trong tuần 2:

Tuần thứ hai tập trung mở rộng kiến thức AWS từ việc triển khai Cloud cơ bản sang tìm hiểu về kiến trúc Compute và Networking.

Tôi đã hoàn thành Module 3, Module 4 và Module 5 của khóa AWS Cloud Practitioner Essentials. Qua đó, tôi hiểu rõ hơn về các mô hình triển khai ứng dụng khác nhau như Virtual Machine truyền thống, Container và Serverless Computing thông qua AWS Lambda.

Module Networking giúp tôi hiểu cách các tài nguyên AWS được tổ chức bên trong môi trường Cloud. Các khái niệm như Region, Availability Zone, VPC, Subnet, Security Group và Network ACL trở nên rõ ràng hơn khi được thực hành trực tiếp trên AWS Console.

Trong phần thực hành, tôi đã tạo lại môi trường EC2 Ubuntu, ôn tập quá trình triển khai Node.js Backend bằng PM2 và tạo một Lambda Function đơn giản để so sánh sự khác biệt giữa mô hình Server truyền thống luôn hoạt động và mô hình Serverless.
