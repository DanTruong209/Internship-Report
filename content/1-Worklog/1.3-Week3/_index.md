## Week 3 Achievements:

This week focused on understanding how AWS manages and stores application data.

I completed Module 6 and Module 7 of AWS Cloud Practitioner Essentials, gaining a clearer understanding of different storage models, including block storage (Amazon EBS), object storage (Amazon S3), and file storage (Amazon EFS).

Before this week, I mainly focused on compute services such as EC2 and Lambda. After learning about AWS storage and database services, I understood how applications separate processing resources and data storage in cloud architecture.

I also studied database approaches including relational databases and NoSQL databases. Concepts such as Amazon RDS, DynamoDB, and caching solutions helped me understand how different database technologies are selected depending on application requirements.

For practical experience, I created an Amazon S3 Bucket on AWS Console, uploaded test files, reviewed object management operations, and explored basic access control concepts.
